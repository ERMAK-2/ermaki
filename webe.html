Тег <style>

C помощью этого тега можно вставлять CSS в HTML. Например: 

<html>
<head>
<style>
CSS-код
</style>
</head>
<body>
</body>
</html>

Тег <script>

С помощью этого тега можно добавлять скрипт JavaScript, C# и других языков. Например: <html> <head> </head> <body> <script> Какой-то код... </script> </body> </html>
